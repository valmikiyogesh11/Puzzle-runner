# 🧩 Puzzle Runner Game

An offline-capable progressive puzzle game inspired by the Chrome Dino concept.
Built using **HTML, CSS, JavaScript + PWA (Progressive Web App)**.

---

## 🚀 Live Demo

👉 (https://puzzlerunner.netlify.app/)

---

## 🎮 Features

* 🎲 Random puzzle generation (math, logic, patterns, alphabet)
* ⏱ Timer-based gameplay (increasing difficulty)
* 🔥 Streak system (continuous correct answers)
* 🏆 High score tracking (saved in browser)
* ❌ Wrong answer resets game
* 📱 Installable as an app (PWA)
* 🚫 Works offline after first load

---

## 🧠 Game Categories

* 🧮 Math puzzles
* 🔢 Number sequences
* 🔠 Alphabet logic
* 🧠 General logic questions
* 🧩 Pattern problems

1. Open the game in browser
2. Click **"Add to Home Screen"**
3. Play anytime (even offline)
